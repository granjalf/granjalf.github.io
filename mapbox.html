Map box

Description:

This project was created to use some libraries to insert map location in a web page.

Detailed Description:

To try adding a map and geolocation information inside a webpage I used UGEL information for the geolocalization of schools in Cusco. Besides using a geolocalization library to display this information I used to investigate how to display big data stracted from a json file without making the browser to freeze.
